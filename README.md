# Subscription Tracker App Store Pages

Static GitHub Pages website for App Store Connect public URLs.

## Files

- `index.html` - small landing page
- `privacy.html` - Privacy Policy URL for App Store Connect
- `support.html` - Support URL for App Store Connect
- `styles.css` - shared styling

## Privacy

This website is intentionally static:

- No analytics
- No cookies
- No tracking pixels
- No external scripts
- No backend

## GitHub Pages URLs

After publishing with GitHub Pages, the URL format is:

```text
https://YOUR-GITHUB-USERNAME.github.io/YOUR-REPOSITORY-NAME/
```

Recommended App Store Connect URLs:

```text
Privacy Policy URL:
https://YOUR-GITHUB-USERNAME.github.io/YOUR-REPOSITORY-NAME/privacy.html

Support URL:
https://YOUR-GITHUB-USERNAME.github.io/YOUR-REPOSITORY-NAME/support.html
```

Replace `h.h0101@yahoo.com` in `privacy.html` and `support.html` before publishing.
